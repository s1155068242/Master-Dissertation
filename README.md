The Python code reproduces Figure 4.2 (pg. 35), which shows the best, median, and worst LISA mock catalogs, and Figure 4.3, which presents the parameter constraints from LISA + supernova data.

For the first case, we follow pg. 30, using the sensitivity of the LISA gravitational wave observatory to generate the number of SS events (mock data) across different redshifts with sigma bar. We then compute the parameter constraints from supernova data using the grid method.

To obtain tighter constraints, we incorporate the LISA mock data generated earlier, compute the Fisher information matrix, and combine the results with the supernova constraints.
